## 기능 목록

- 음식점 카테고리별로 필터링
- 음식점 이름순, 거리순으로 정렬
- 음식점 상세 정보 모달

## 실행 방법

1. 레포지토리 클론

```bash
git clone -b leejin-yang --single-branch https://github.com/woowacourse/react-lunch.git
```

2. 의존성 설치

```bash
npm install
```

3. 실행

```bash
npm run start
```
